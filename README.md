# Philadelphia Presbyterian Church Website

This is a simple static church website.

## Files

- `index.html` — website content
- `styles.css` — design and layout
- `script.js` — mobile menu and current year

## How to edit

Open `index.html` and replace the placeholder text for:

- Meeting address
- Contact email
- Parking details
- Sermon links
- Church history
- Leadership information

## Free hosting options

You can upload these files to GitHub Pages, Cloudflare Pages, Netlify, or Vercel.

For the simplest free route, create a GitHub repository, upload these files, then turn on GitHub Pages in the repository settings.
